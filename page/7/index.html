使用winscp实现Window系统和Linux系统的文件目录操作

发表于 2016-05-23   |   字数统计: 211(字)   |   阅读时长: 1(分) | 分类于 tool |

简介

winscp是一个Windows环境下使用SSH的开源图形化SFTP客户端。同时支持SCP协议。它的主要功能就是在本地与远程计算机间安全的复制文件。
参考链接：http://baike.baidu.com/item/winscp?fr=aladdin

使用步骤

  1. 网络上查找，下载winscp。
  2. winscp是一个绿色软件，不需要安装可直接使用

  3. 在目录中找到winscp可执行文件，双击，进入主界面

  4. 输入要连接的linux电脑的IP地址、FTP端口、登录的用户名和密码信息，点击登录

  5. 登录成功后，可看到两个计算机的目录 结构，左边为本机，右边为远程的Linux计算机的
  6. 此时就可以将windows中的文件上传到linux中
  7. 或者将linux中的文件下载到windows中

Spring配置文件的版本与jar包不一致，导致项目启动失败

发表于 2016-05-19   |   字数统计: 295(字)   |   阅读时长: 1(分) | 分类于 spring |

问题

公司网络不稳定，当断网的时候，重启web项目时，项目启动失败，提示出了如下的错误：

分析原因

上述错误因为Spring的xsd文件spring-beans-3.0.xsd没有获取到，无法连接到Spring的官网上去请求到此文件。
查询资料，了解到Spring首先会到本地项目的Spring的jar中查找xsd文件，如果要查找的文件的版本高于jar的版本，那么将远程到官网中进行请求文件。
因为web项目采用的Spring版本是2.5，而在Spring配置文件中声明的是3.0版本。所以会远程到官网中请求文件，但目下处于断网状态，请求不成功。

解决办法

将项目的Spring配置文件声明中的3.0版本修改为2.5版本，与jar包保持一致，即可解决此问题。

SQL的UNION与UNION ALL的差别

发表于 2016-05-14   |   字数统计: 65(字)   |   阅读时长: 1(分) | 分类于 sql |

在SQL语句中，UNION和UNION ALL都可用于合并结果集。
两者的最主要的差别是：

  • 使用UNION，会剔除合并结果集中的重复的数据
  • 使用UNION ALL，则会保留合并结果集中的重复的数据
